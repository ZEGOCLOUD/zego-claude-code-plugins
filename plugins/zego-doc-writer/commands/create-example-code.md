请使用 `create-example-code` skill 为用户创建 ZEGO 产品示例代码。

该 skill 会：
1. 收集用户详细需求
2. 引导选择目标产品（音视频 SDK、实时消息、即时通讯等）
3. 确认开发端（客户端/服务端/两者都要）
4. 选择平台和语言（Android/iOS/Web/Node.js/Python 等）
5. 输出交互稿并确认
6. 查阅相关文档并生成架构设计
7. 逐步实现示例代码
8. 构建测试并输出交付文档
