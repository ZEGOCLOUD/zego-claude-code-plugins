# QuickStart（快速开始）结构标准

## 必需章节

| 章节 | 要求 |
|------|------|
| **简介** | 说明本文档将指导用户完成什么（如"5 分钟实现音视频通话"） |
| **前提条件** | 说明开始前需要准备的账号、权限、设备等 |
| **集成 SDK** | 说明如何集成 SDK（CocoaPods/Maven/Gradle/NPM 等） |
| **实现核心功能** | 分步骤说明实现核心功能的代码流程，包含完整代码示例 |
| **运行项目** | 说明如何编译运行，验证功能是否正常 |
| **下一步** | 引导用户查看进阶功能文档 |

## 检查要点

- 前置条件是否完整
- SDK 集成步骤是否准确
- 代码示例是否完整可运行
- 是否有预期效果说明

## 输出格式

```markdown
## QuickStart 结构检查结果

### ✅ 结构完整
（所有必需章节都存在时）

### ❌ 缺失章节

1. **[章节名称]** - 缺失
   建议：[具体的改进建议]

### ⚠️ 章节内容不足

1. **[章节名称]**
   - 当前内容：[描述当前状态]
   - 建议：[具体的改进建议]
```
